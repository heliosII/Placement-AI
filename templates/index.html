About Project

PlacementAI is a Machine Learning web application that predicts whether a student is likely to be placed based on multiple academic and technical features.

The prediction model uses five important attributes:

  • CGPA
  • IQ
  • Internship Experience
  • Number of Projects
  • DSA Rating

The model is trained using Scikit-Learn Logistic Regression with StandardScaler preprocessing and deployed using Flask.

Made with ❤️ using Flask, HTML, CSS, JavaScript & Scikit-Learn
